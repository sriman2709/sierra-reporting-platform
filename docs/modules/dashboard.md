# Dashboard Overview

The Dashboard is the platform's home page and entry point to all modules.

## Sections

### Platform Banner
Shows platform name, description, and live SAP HANA Cloud connectivity confirmation. The tags at the bottom summarize platform capabilities (HANA tables, modules, AI agents, auth model).

### Cross-Domain KPI Strip
10 live KPI tiles pulled from multiple modules simultaneously:

| KPI | Source Module |
|---|---|
| Active Grants | Grants Management |
| Total Grant Funding | Grants Management |
| Fund Balance | Fund Accounting |
| Available-to-Spend | Fund Accounting |
| Cash Position | Treasury & Revenue |
| Active Contracts | Procurement & AP |
| Capital Projects | Capital Projects |
| Active Employees | HR & Workforce |
| Fleet Vehicles | Fleet Management |
| Stock Alerts | Inventory & Warehouse |

All values are live from SAP HANA Cloud — no caching.

### Module Cards
14 clickable module cards — one per operational module. Each card shows:

- Module icon
- Module name
- Brief description of what the module covers
- A phase badge (Financial / Program / Operations / Workforce) in the top-right corner
- Click anywhere on the card to open the module

### AI & Agents Quick-Launch
Two prominent cards at the bottom linking to:

- **Agent Hub** — Run any of the 4 autonomous AI agents
- **Sierra Intelligence** — Open the AI chat analyst

### Architecture Note
Technical footer showing the platform stack and quick links to the Public Portal and Roadmap.
